Der Insertionsort

Der Insertionsort zählt, wie der Selectionsort, zu den natürlichen Sortieralgorithmen.


Verfahren


Angenommen das obere Array soll nun sortiert werden.

Zunächst wählt man das erste Element (2) und sortiert es soweit nach rechts,
bis das Element rechts daneben größer ist.


Jetzt wählt man immer das erste unsortierte Element und verfährt genauso,



bis das Array vollständig sortiert ist.



Es gibt auch eine Variante, in der die Elemente nach links einsortiert werden.
